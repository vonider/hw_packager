Программа создает документ Microsoft Word, в котором указывает название дисциплины,
ваши данные и вставляет фотографии из директории photos. Программа позволяет быстро
создать файл с домашней работой. Это экономит время на дистанционном обучении.

Для запуска программы необходимо установить библиотеки из файла requirements.txt.
Укажите ваше имя, фамилию и группу в файле config.txt.
